# @rush_ui/tokens

Rush UI 的设计 token 包，提供颜色、圆角、间距和阴影等基础值。

## 安装

```bash
pnpm add @rush_ui/tokens
```

## 使用

```ts
import { tokens } from "@rush_ui/tokens";

const cardRadius = tokens.radius.md;
const pageBackground = tokens.color.canvas;
```

完整 token 定义参见 [Rush UI 仓库](https://github.com/TwilighRush/rush-ui)。
